FAQ

Preguntas frecuentes

¿Cómo instalo PISKU? ▼
Necesitás Python 3.11+. Corrés pip install pisku y ya tenés el comando pisku disponible globalmente. Igual que instalás black, ruff, o cualquier tool Python.
¿Cómo funciona la activación PRO? ▼
Comprás PRO con Stripe. Recibís una license key por email tipo PISKU-PRO-XXXX-XXXX. La activás con pisku activate-pro <key>. El CLI valida la key contra nuestro servidor y listo — tier PRO activo. La validación se cachea localmente, así que funciona offline después.
¿Qué pasa si cancelo PRO? ▼
Volvés a FREE automáticamente cuando expira tu licencia. No perdés tus skills ni tu historial — solo los features PRO quedan bloqueados. Todos tus archivos .md y el config.json quedan intactos.
¿Los skills son genéricos o puedo crear los míos? ▼
Son tuyos. Un skill es simplemente un archivo .md en la carpeta skills/. Podés escribir los tuyos con pisku add-skill mi-skill --category backend, o copiar cualquier archivo Markdown que ya tengas. Se versiona con git sin problema.
¿Funciona con cualquier LLM? ▼
Sí. PISKU genera un archivo .md con el contexto seleccionado. Ese archivo lo pegás donde quieras: Claude, GPT, Gemini, tu propio script con la API. No estás atado a ningún proveedor.
